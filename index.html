Google Sheets sync

1Create your Google Sheet

Open Google Sheets and create a new spreadsheet. Name it Golden Circle Tracker. Make sure headers in row 1 are: unit, week, group, amount, notes, timestamp

2Create a Google Apps Script

In your Sheet, go to Extensions → Apps Script. Delete any code there and paste this:

function doPost(e){ var ss = SpreadsheetApp.getActiveSpreadsheet(); var sh = ss.getSheetByName("Sheet1") || ss.getActiveSheet(); var d = JSON.parse(e.postData.contents); if(d.action === "write"){ sh.appendRow([d.unit,d.week,d.group,d.amount,d.notes,d.timestamp]); return ContentService.createTextOutput(JSON.stringify({ok:true})).setMimeType(ContentService.MimeType.JSON); } if(d.action === "read"){ var rows = sh.getDataRange().getValues(); return ContentService.createTextOutput(JSON.stringify({rows:rows})).setMimeType(ContentService.MimeType.JSON); } }

3Deploy as Web App

Click Deploy → New deployment → Web app. Set "Execute as" to Me and "Who has access" to Anyone. Copy the Web App URL and paste it below.
